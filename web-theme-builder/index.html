Additional Number Row

This controls only the optional number row colorway. It does not turn the number row on or off.

Default reading is outer digits 1 2 3 8 9 0 as alpha and center digits 4 5 6 7 as mod, but any alpha/mod/accent split can be selected.

Dingul Color Roles

Alpha and mod define the base colorway. Mod inv is only for a real third accent tone; do not use it just to manufacture a 3-tone theme.

Role-first mapping guide

  1. 1. Global roles: set alphaKey, modifierKey, accentKey, accent, and secondary first.
  2. 2. Tone count: decide whether the image is 2-tone or 3-tone. For 2-tone themes, keep accentPolicy empty and do not force modInv as an accent.
  3. 3. Dingul roles: mirror the same colorway through dingulColors.alpha and dingulColors.mod; use dingulColors.modInv only for a real third tone.
  4. 4. Number row: start from half_mod_4567 unless the image clearly uses another number row split.
  5. 5. Accent policy: if a whole visual group is visibly accented, use accentPolicy targets instead of individual keys.
  6. 6. Per-key fallback: use keyTextColorOverrides or keyBackgroundColorOverrides only for isolated keys that cannot be expressed by roles or policy.
